# GPA计算器

一个功能强大的GPA（平均学分绩点）计算器，支持输入课程分数自动计算GPA和加权平均分。

## 功能特点

### 🎯 核心功能
- ✅ 添加多门课程（课程名称、学分、分数）
- ✅ 支持修改已添加的课程信息
- ✅ 支持删除课程
- ✅ 自动计算GPA（平均学分绩点）
- ✅ 自动计算加权平均分
- ✅ 实时显示课程列表
- ✅ 响应式设计，支持移动端

### 📊 成绩计算规则
- **90分及以上**：绩点为 **4.0**（满绩点）
- **60-89分**：绩点为 **1.0 + (分数-60) × 0.1**
  - 例如：60分 = 1.0绩点，70分 = 2.0绩点，80分 = 3.0绩点
- **60分以下**：绩点为 **0.0**

## 使用方法

### 添加课程
1. 在表单中填写 **课程名称**、**学分**、**分数**
2. 点击 **添加课程** 按钮
3. 课程会自动添加到课程列表中

### 修改课程
1. 在课程列表中找到需要修改的课程
2. 点击课程对应的 **修改** 按钮
3. 表单会自动填充该课程的信息
4. 修改需要更改的字段
5. 点击 **更新课程** 按钮保存更改

### 删除课程
1. 在课程列表中找到需要删除的课程
2. 点击课程对应的 **删除** 按钮
3. 课程会从列表中移除

### 计算GPA
1. 添加完所有课程后
2. 点击 **计算** 按钮
3. 系统会显示 **GPA**、**加权平均分** 和 **总学分**

## 示例

### 添加课程

| 课程名称 | 学分 | 分数 | 绩点  |
|---------|------|------|-------|
| 高等数学 | 4.0  | 85   | 3.5   |
| 英语     | 3.0  | 92   | 4.0   |
| 计算机基础 | 2.5 | 78   | 2.8   |

### 计算结果

- **GPA**: 3.47
- **加权平均分**: 84.89
- **总学分**: 9.5

## 注意事项

1. 分数必须在 **0-100** 之间
2. 学分必须大于 **0**
3. 确保后端服务器正在运行
4. 支持小数分数输入（例如：85.5）
5. 支持小数学分输入（例如：3.5）
